get-attr {purrr}R Documentation

Infix attribute accessor

Description

Please use the %@% operator exported in rlang. It has an interface more consistent with @: uses NSE, supports S4 fields, and has an assignment variant.

Usage

x %@% name

Arguments

x

Object

name

Attribute name

Examples

factor(1:3) %@% "levels"
mtcars %@% "class"
