# Data Dictionary: Contracts

A dataframe containing the data dictionary for
[`load_contracts()`](https://nflreadr.nflverse.com/reference/load_contracts.md)

## Usage

``` r
dictionary_contracts
```

## Format

An object of class `data.frame` with 15 rows and 3 columns.
