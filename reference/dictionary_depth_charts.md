# Data Dictionary: Depth Charts

A dataframe containing the data dictionary for
[`load_depth_charts()`](https://nflreadr.nflverse.com/reference/load_depth_charts.md)

## Usage

``` r
dictionary_depth_charts
```

## Format

An object of class `data.frame` with 12 rows and 3 columns.
