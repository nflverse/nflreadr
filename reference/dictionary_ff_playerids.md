# Data Dictionary: Fantasy Player IDs

A dataframe containing the data dictionary for
[`load_ff_playerids()`](https://nflreadr.nflverse.com/reference/load_ff_playerids.md)

## Usage

``` r
dictionary_ff_playerids
```

## Format

An object of class `data.frame` with 35 rows and 3 columns.
