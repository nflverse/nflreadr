# Data Dictionary: Fantasy Football Rankings

A dataframe containing the data dictionary for
[`load_ff_rankings()`](https://nflreadr.nflverse.com/reference/load_ff_rankings.md)

## Usage

``` r
dictionary_ff_rankings
```

## Format

An object of class `data.frame` with 25 rows and 3 columns.
