# Data Dictionary: Player Stats

A dataframe containing the data dictionary for
[`load_player_stats()`](https://nflreadr.nflverse.com/reference/load_player_stats.md)

## Usage

``` r
dictionary_player_stats
```

## Format

An object of class `data.frame` with 114 rows and 3 columns.
