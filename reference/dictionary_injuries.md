# Data Dictionary: Injuries

A dataframe containing the data dictionary for
[`load_injuries()`](https://nflreadr.nflverse.com/reference/load_injuries.md)

## Usage

``` r
dictionary_injuries
```

## Format

An object of class `data.frame` with 16 rows and 3 columns.
