# nflreadr: Download 'nflverse' Data

A minimal package for downloading data from 'GitHub' repositories of the
'nflverse' project.

## See also

Useful links:

- <https://nflreadr.nflverse.com>

- <https://github.com/nflverse/nflreadr>

- Report bugs at <https://github.com/nflverse/nflreadr/issues>
