# Data Dictionary: Team Stats

A dataframe containing the data dictionary for
[`load_team_stats()`](https://nflreadr.nflverse.com/reference/load_team_stats.md)

## Usage

``` r
dictionary_team_stats
```

## Format

An object of class `data.frame` with 102 rows and 3 columns.
